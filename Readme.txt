Welcome to Flask App
Enter your file name with extension (from file1.txt, file2.txt, file3.txt, file4.txt)
Enter Strating and Ending lines of a file (Optional)
Note : Give Input in the URL

Example : http://127.0.0.1:5000/file1/10/20

It will print data in file1 from 10 to 20 lines.

If you enter any other file otherthan files mentioned above it will raise error.


Download the code from github then open it and follow the below steps : 

Installing process :

Step 1 : 

pip install virtualenv

Step 2 : 

virtualenv venv

Step 3 : 

.\venv\Scripts\activate
Then your virtual environment will be activated
 
Step 4 : 

pip install flask


Step 5 :

flask run
